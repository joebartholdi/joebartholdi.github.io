- Signal Science Automation Script: Crafted a Python script to fetch web request data from the Signal Sciences API for a specific site over the last 24 hours. It calculates the start and end times to cover the previous full day, rounded to the nearest hour, ensuring the data fetched is for a precise and recent timeframe. Utilizing the Signal Sciences API, it systematically collects web request data, handling the iteration by following the 'next' URL provided in each API response until all data within the timeframe is retrieved. The aggregated data is then compiled into a DataFrame using Pandas, which outputs this collected data into a CSV file. The file is named uniquely based on the site name and the time range covered, providing a structured and accessible record of web request logs for further analysis or archival purposes. The information obtained from this script includes the timestamp, client IP address, requested URL, HTTP method, response status code, user agent, referring URL, and server response time. This information is important for analysts to monitor for anomalies and other cyber threats.
